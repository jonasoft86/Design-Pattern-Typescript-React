## Storybook 跑起來

執行`yarn story`後，開啟`http://localhost:6006`，然後切到`Iterator Pattern/CustomerApp`，就可以看到畫面了。

先加入今天要寄的潛在客戶資訊到資料列表中，然後點擊產生信封資訊與紀錄，就會執行迭代器，依序輸出處理過後的資料。
